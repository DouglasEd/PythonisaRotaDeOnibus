Pythonisa
Já pensou acompanhar o trajeto dos ônibus em Macapá?
Quem somos?
O projeto Pythonisa é um projeto de pesquisa da Unifap, financiado pelo CNPQ, que visa monitorar ônibus com o uso de visão computacional para permitir o monitoramento dos ônibus de Macapá e realizar a predição dos horários para os usuários.
